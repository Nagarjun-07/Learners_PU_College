# LGS Educational Website Design Guide

## Design Philosophy

### Color Palette
- Primary: Deep Navy (#1e3a8a) - Professional, trustworthy, academic
- Secondary: Warm Orange (#f97316) - Energy, enthusiasm, achievement
- Accent: Soft Blue (#3b82f6) - Technology, innovation, learning
- Neutral: Warm Gray (#6b7280) - Balance, sophistication
- Background: Clean White (#ffffff) - Clarity, openness

### Typography
- Display Font: "Playfair Display" - Elegant serif for headings, academic prestige
- Body Font: "Inter" - Clean sans-serif for readability and modern feel
- Accent Font: "Source Code Pro" - Technical font for course codes and data

### Visual Language
- Clean, professional aesthetic that builds trust with parents
- Warm, approachable design that appeals to students
- Academic excellence conveyed through sophisticated typography
- Modern educational technology integration
- Accessibility-first design principles

## Visual Effects & Animations

### Used Libraries
- Anime.js: Smooth entrance animations and micro-interactions
- ECharts.js: Data visualization for student performance and achievements
- Splide.js: Image carousels for campus tours and events
- p5.js: Interactive background elements and creative coding
- Pixi.js: Advanced visual effects for hero sections
- Matter.js: Physics-based animations for engaging interactions

### Effect Implementation
1. **Hero Section**: Subtle parallax scrolling with floating educational icons
2. **Course Cards**: 3D tilt effect on hover with depth shadows
3. **Faculty Profiles**: Smooth expand/collapse animations with staggered timing
4. **Achievement Gallery**: Filter transitions with smooth opacity and scale changes
5. **Navigation**: Gentle underline animations and color transitions
6. **Background**: Soft geometric patterns that respond to scroll position

### Header Effect
- Clean navigation bar with subtle drop shadow
- Smooth color transitions on scroll
- Logo animation on page load
- Mobile-responsive hamburger menu with smooth transitions

### Styling Approach
- Consistent 8px grid system for spacing
- Subtle shadows for depth without distraction
- Rounded corners (8px) for modern, friendly feel
- High contrast ratios for accessibility (4.5:1 minimum)
- Responsive breakpoints: 768px (tablet), 1024px (desktop)

### Interactive Elements
- Buttons: Gentle scale transform on hover (1.05x)
- Cards: Lift effect with increased shadow
- Forms: Focus states with color transitions
- Links: Underline animation from left to right
- Images: Subtle zoom effect on hover (1.1x scale)
